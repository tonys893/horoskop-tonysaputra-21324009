Aplikasi Horoskop Sederhana PHP

Aplikasi ini dibuat menggunakan bahasa pemrograman PHP.
Program ini digunakan untuk menentukan zodiac berdasarkan tanggal lahir yang diinput oleh user.

Konsep yang digunakan:
- Form HTML
- Method POST
- Variabel PHP
- Percabangan if else
- Fungsi date()
- Output echo

Cara menjalankan:
1. Install XAMPP
2. Jalankan Apache
3. Simpan folder di htdocs
4. Buka localhost/horoskop
